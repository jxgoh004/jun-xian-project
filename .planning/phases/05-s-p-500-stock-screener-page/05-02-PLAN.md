5. **Column header click:** Add click handler to each `<th data-col>`. On click: if same column, toggle `sortDir` between 1 and -1. If different column, set `sortCol` and `sortDir = -1`. Update `data-sort` attribute on `<th>` elements (remove from all, set `asc` or `desc` on active). Call `renderTable()`.

6. **`getFiltered()` function (per D-09, D-10, D-11):**
   ```javascript
   function getFiltered() {
     return allStocks.filter(function(s) {
       var q = filterText.toLowerCase();
       var matchText = !q || s.ticker.toLowerCase().indexOf(q) !== -1 || s.company_name.toLowerCase().indexOf(q) !== -1;
       var matchSector = !filterSector || s.sector === filterSector;
       return matchText && matchSector;
     }).sort(function(a, b) {
       var av = a[sortCol], bv = b[sortCol];
       if (av === null && bv === null) return 0;
       if (av === null) return 1;
       if (bv === null) return -1;
       if (typeof av === 'string') return sortDir * av.localeCompare(bv);
       return sortDir * (av < bv ? -1 : av > bv ? 1 : 0);
     });
   }
   ```

7. **`renderTable()` function:** Clear `#table-body`. For each stock in `getFiltered()`, create a `<tr>` using `document.createElement`. Each cell uses `td.textContent` for text values (per research anti-pattern guidance). For Discount % column: show as `"18.0%"` or `"—"` if null. For Price and Intrinsic Value: show as `"$172.50"` or `"—"` if null. For Method: show `"OCF"` or `"FCF"` (uppercase). For Valuation column: create a `<span>` with class `badge badge-{slug}` where slug is the label lowercased with spaces replaced by hyphens (e.g., `"Highly Undervalued"` -> `"highly-undervalued"`). Row click calls `navigateToCalculator(stock.ticker)`. Update `#row-count` text: `"Showing X of Y stocks"`.

8. **`navigateToCalculator(ticker)` function (per D-16):**
   ```javascript
   function navigateToCalculator(ticker) {
     window.parent.postMessage({ type: 'navigate', project: 'calculator', ticker: ticker }, '*');
   }
   ```

9. **Debounce helper:**
   ```javascript
   function debounce(fn, ms) {
     var timer;
     return function() {
       clearTimeout(timer);
       timer = setTimeout(fn.bind(this), ms);
     };
   }
   ```
   Apply to search input event: `searchInput.addEventListener('input', debounce(function() { filterText = this.value; renderTable(); }, 150));`

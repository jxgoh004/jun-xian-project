**Step 2 — Replace img tag with picture element in docs/index.html (D-14).**

Find the card's img element (currently inside the `.card` div in `#home-view`):
```html
        <img src="projects/calculator/thumbnail.png" alt="Intrinsic Value Calculator screenshot">
```

Replace it with:
```html
        <picture>
          <source srcset="projects/calculator/thumbnail.webp" type="image/webp">
          <img
            src="projects/calculator/thumbnail.png"
            alt="Intrinsic Value Calculator screenshot"
            loading="lazy"
            width="1854"
            height="771">
        </picture>
```

The `width="1854"` and `height="771"` attributes match the actual PNG dimensions (1854×771px per research file). These prevent layout shift (CLS) by letting the browser reserve the correct aspect ratio before the image loads.
  </action>

The on-disk annotated charts in `docs/projects/patterns/charts/` (42 files today) are the source material. Specifically:

- Read `docs/projects/patterns/data.json`, find the first row with `yolo_conf >= 0.50`, take its `chart_path` (e.g. `charts/APD_2026-04-20.png`).
- Use Python + Pillow (already in the project's environment) to open the source PNG, resize to fit a 1200×630 canvas with letterboxing (preserve aspect ratio, do NOT crop), save as `docs/projects/patterns/og-image.png`. Background fill: `#080c12` (analytical-dark) to match the brand.

Example Python snippet to run (activate the project venv per MEMORY.md — `.venv\Scripts\activate` on Windows):
```python
from PIL import Image
import json, os
data = json.load(open('docs/projects/patterns/data.json'))
green = next((d for d in data['detections'] if (d.get('yolo_conf') or 0) >= 0.50), None)
src_path = os.path.join('docs/projects/patterns', green['chart_path'])
src = Image.open(src_path).convert('RGB')
target = Image.new('RGB', (1200, 630), '#080c12')
# Fit src inside 1200x630 preserving aspect ratio
src.thumbnail((1180, 610), Image.LANCZOS)
x = (1200 - src.width) // 2
y = (630 - src.height) // 2
target.paste(src, (x, y))
target.save('docs/projects/patterns/og-image.png', 'PNG', optimize=True)
print(f"Wrote og-image.png from {src_path} ({src.width}x{src.height} → 1200x630)")
```

Save this script as a one-shot inline command (not a committed scripts/* file — this is a one-time generation step). Run from the repo root with venv active.
